Setup
To run any of these projects, simply clone the repository and compile the respective C files using a C compiler (e.g., GCC).

bash
Copy code
git clone https://github.com/Siddarthva/Projects_using_C.git
cd Projects_using_C
gcc project_file.c -o project_output
./project_output
